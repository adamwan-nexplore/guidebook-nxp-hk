## What is delivery pipelines

It is a bunch of programs to run to VERIFY 

## How the good pipeline looks like
- Fast enough (5 mins)
- Cover the essential parts 
    - system behaviour test cases
    - security check
    - styles & comments format
    - test build
    - test data migration behaviour
- Provide a consistent and transparent way to notify the status

- Automate the standard build procedures
- Automate the standard deploy procedures
